THE CRANKS

[View site]

The Cranks are an alternative rock band, hailing from Massachusetts in the United States. This project involved the revamp of an existing website, using a modern platform – one that is responsive and mobile-friendly. For this task, the front-end framework, Bootstrap, was used as the foundation for the new website. Due to the nature of the existing website, the project was designed and developed from scratch, whilst maintaining the current content and styles. The end result was a website that looks modern, is responsive, and is easier to use.

This project took place in 2015, and was completed remotely. After an initial meeting on Skype, expectations for the new website were laid out, and requirements were listed. The requirements were quite straightforward – retain the existing website style, and make the website responsive.

To complete this task, Bootstrap was used. As a pioneer in front-end website frameworks, Bootstrap was a suitable choice to build the new website with, as it contained all of the required components, and was easy to explain to those who may not be familiar with the framework.

As the project progressed, content was placed within percentage-based columns – a popular method of responsive website design. This method allows the website to scale appropriately, depending on the device being used to view the website.

Mobile view of new website

The previous version of the website was quite a mess, with content overlapping in several areas. Because of this, content was difficult to view, even on desktop environments. The website was also performing very slowly. The files being used were not optimized, with a separate CSS file used for each individual page. This resulted in massive overhead, to the tune of 50 megabytes. Change was definitely necessary.

After utilizing the Bootstrap framework, as well as HTML and CSS, the new website was beginning to come together. By shifting the styles from each CSS file into one compiled file, the previous overhead was lessened, and the website began to load much faster. In the end, the entire website weighed in at just under 7 megabytes.

Eventually, the revamp was completed, resulting in a swift website that is modern, responsive, and easy to use.
